# PressWatch

PressWatch は、環境省の報道発表を収集・整理・検索しやすくする個人開発プロジェクトです。

前職で培ったドメイン知識を活かし、現場で実際に起こる情報収集負担や見逃しリスクを軽減することを目的としています。

## 現在の開発状況

現在は、MVP 開発に向けた初期ドキュメントと技術方針を整備している段階です。

## 技術スタック

- フロントエンド: Next.js / React / TypeScript
- バックエンド: FastAPI / Python
- データベース: PostgreSQL
- 開発環境: Docker / docker compose
- パッケージ管理:
  - フロントエンド: pnpm
  - Python: uv
